# Expected Behavior Matrix

| Scenario | Expected V3 Behavior |
|---|---|
| Only knowledge base uploaded | Say actual RFP is missing; do not appraise |
| Actual RFP uploaded | Run full appraisal workflow |
| Field-work language found | Classify field-work level and explain risk |
| Surveying or LiDAR found | Flag possible subcontractor need |
| Award criteria missing | Mark as missing; do not invent scoring weights |
| Page limit missing | Mark as not found or unknown |
| Project location missing | Mark geography risk as unknown |
| Insurance terms found | Flag for leadership/admin review |
| Historical examples available | Compare but do not copy past decision |
| Scope unclear | Conditional Go or needs review |
| Heavy out-of-scope field work | No-Go or high-risk Conditional Go |
| Good scope but missing value | Conditional Go until value is confirmed |
