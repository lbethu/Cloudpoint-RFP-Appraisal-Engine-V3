```text

Security / Compliance Risk Thresholds:

Page-Limit Strategy Rules:

Award Criteria Preferences:

Historical Go Examples:

Historical No-Go Examples:

Historical Won Examples:

Historical Lost Examples:

Required Output Format:

Tone / Writing Style:
```

## How To Use

Replace the Cloudpoint-specific details in the master prompt with the values above.

Do not remove:

- missing RFP detection
- document classification
- field-work detection
- subcontractor need detection
- compliance matrix
- scorecard logic
- final recommendation logic
