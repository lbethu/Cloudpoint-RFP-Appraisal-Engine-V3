# Future Roadmap

## Purpose

This file lists possible improvements for future versions of the RFP Appraisal Engine.

---

## Near-Term Improvements

- Add more sample RFP test cases
- Add better historical Go/No-Go tagging
- Add more detailed page-limit strategy examples
- Add sample PDF-ready reports
- Add example scorecards
- Add a standard RFP intake form

---

## Mid-Term Improvements

- Build a simple web dashboard for RFP intake
- Store RFP review outputs in a structured table
- Add JSON output mode
- Add export-ready PDF report generation
- Add a scoring dashboard
- Add review status tracking

---

## Advanced Improvements

- Create a historical RFP similarity engine
- Add vector search over historical RFPs
- Add approval workflow for Go/No-Go decisions
- Add automated tagging of RFPs after review
- Add dashboard analytics for pursued vs declined RFPs
- Add integration with proposal management tools

---

## Long-Term Vision

The long-term vision is to turn the RFP Appraisal Engine into an internal AI decision-support platform for proposal strategy.

Future system flow:

```text
RFP Upload → AI Appraisal → Human Review → Decision Log → Historical Learning → Better Future Recommendations
```
